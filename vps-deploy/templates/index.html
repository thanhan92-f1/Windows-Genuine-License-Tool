⚡ Cách 1: Sử dụng Tool tự động

Tool sẽ tự động quét và gỡ bỏ tất cả công cụ crack trong một lần chạy.

# Mở PowerShell với quyền Administrator (Windows + X → Terminal Admin) # Chạy lệnh sau: irm https://irm-genuine-license-windows.hitechcloud.vn | iex

Tool sẽ thực hiện:

  • Gỡ Product Key hiện tại (slmgr /upk)
  • Xóa key khỏi Registry (slmgr /cpky)
  • Xóa thông tin KMS server (slmgr /ckms)
  • Reset trạng thái License (slmgr /rearm)
  • Xóa thư mục/file KMS (KMSpico, KMSAuto, KMS-R@1n...)
  • Xóa Scheduled Tasks liên quan KMS
  • Sửa file Hosts (xóa dòng block Microsoft)
  • Khôi phục dịch vụ Windows Update & Software Protection
  • Dọn dẹp Registry keys liên quan KMS
⚠️ Quan trọng: Nếu đang dùng hệ thống Bootrom, hãy Mở Super OS trước khi chạy tool. Sau khi hoàn tất, Đóng Super OS để đồng bộ toàn hệ thống.

🔧 Cách 2: Gỡ thủ công (5 bước)

Dành cho kỹ thuật viên muốn kiểm soát từng bước.

  1. Mở CMD với quyền Administrator

    Nhấn Windows + S, gõ cmd, click chuột phải chọn Run as administrator.

    # Hoặc nhấn Windows + X, chọn "Terminal (Admin)"

  2. Gỡ Product Key hiện tại

    Lệnh này gỡ bỏ hoàn toàn key lậu ra khỏi hệ điều hành:

    slmgr.vbs /upk
    📋 Kết quả: Thông báo "Uninstalled product key successfully"

  3. Xóa key khỏi Registry

    Xóa dấu vết key rác trong Registry:

    slmgr.vbs /cpky

  4. Xóa thông tin KMS & Reset License

    Xóa tên máy chủ KMS ảo và reset trạng thái cấp phép:

    slmgr.vbs /ckms slmgr.vbs /rearm

  5. Sửa file Hosts

    Tools crack thường sửa file hosts để chặn kết nối Microsoft. Cần khôi phục lại:

    # Mở file hosts bằng Notepad (quyền Admin): notepad C:\Windows\System32\drivers\etc\hosts # Xóa tất cả dòng chứa: # activation.microsoft.com # validation.sls.microsoft.com # kms / crack / kmspico # Lưu lại bằng Ctrl + S

  6. Xóa thư mục/file KMS (nếu có)

    Mở File Explorer, bật hiện file ẩn (Hidden items), kiểm tra và xóa:

    C:\Program Files\KMSpico C:\Program Files (x86)\KMSpico C:\ProgramData\KMSAutoS C:\ProgramData\KMSAuto C:\Windows\KMS-R@1n

  7. Xóa Scheduled Tasks

    Mở Task Scheduler (gõ "Task Scheduler" vào Start), tìm và xóa:

    Task NameMô tả
    AutoKMSTự động kích hoạt KMS
    KMSAutoTool KMS Auto
    SvcRestartTaskKhởi động lại dịch vụ KMS
    KMSpicoTool KMSpico

  8. Khởi động lại & Nhập key bản quyền

    Sau khi hoàn tất, khởi động lại máy. Vào Settings → Activation → Change product key để nhập key chính hãng.

    # Hoặc dùng CMD: slmgr.vbs /ipk XXXXX-XXXXX-XXXXX-XXXXX-XXXXX slmgr.vbs /ato

🔍 Cách kiểm tra trạng thái License

Kiểm tra bằng CMD

Mở CMD quyền Admin và chạy:

slmgr.vbs /dlv
📋 Kết quả mong muốn:
✅ Sạch: "Licensed" hoặc "Notification"
❌ Có crack: "KMS Client" hoặc "Volume activation expiration: 180 days"

Kiểm tra nhanh bằng PowerShell

# Kiểm tra trạng thái kích hoạt: (Get-WmiObject -query "select * from SoftwareLicensingProduct where PartialProductKey is not null").LicenseStatus # 0 = Unlicensed, 1 = Licensed, 2 = OOB Grace, 3 = OOT Grace, 4 = Non-genuine # Kiểm tra có phải KMS client không: slmgr.vbs /dli | Select-String "KMS"

Kiểm tra Scheduled Tasks đáng ngờ

# Liệt kê tất cả tasks liên quan KMS: schtasks /query /fo LIST | findstr /i "kms auto crack"

Kiểm tra file hosts

# Xem nội dung file hosts: type C:\Windows\System32\drivers\etc\hosts # Nếu có dòng chứa "kms", "crack", "activation.microsoft.com" # → Hệ thống đang bị can thiệp

Kiểm tra thư mục KMS

# Kiểm tra các thư mục KMS có tồn tại không: dir "C:\Program Files\KMSpico" 2>nul dir "C:\ProgramData\KMSAutoS" 2>nul dir "C:\Windows\KMS-R@1n" 2>nul # Nếu thấy thư mục → Cần xóa ngay

Bảng tra cứu kết quả

Trạng tháiÝ nghĩaHành động

❓ Câu hỏi thường gặp

Tool có xóa game / dữ liệu không?

Không. Tool chỉ gỡ bỏ các công cụ crack (KMS, registry, scheduled tasks). Toàn bộ dữ liệu cá nhân, game, phần mềm khác vẫn được giữ nguyên.

Sau khi gỡ có cần cài lại Windows không?

Không cần. Chỉ cần nhập key bản quyền mới và kích hoạt lại là được. Windows sẽ hoạt động bình thường.

Tool có an toàn không?

Hoàn toàn an toàn. Tool chỉ thực hiện các lệnh chuẩn của Windows (slmgr.vbs) và xóa các file/thư mục liên quan đến phần mềm crack. Mã nguồn mở, có thể kiểm tra tại GitHub.

Phòng net Bootrom thì làm sao?

Bước 1: Mở Super OS (băng image) trên 1 máy trạm đại diện.
Bước 2: Chạy tool trên máy trạm đó.
Bước 3: Đóng Super OS (lưu image).
Tất cả máy trạm sẽ được đồng bộ.
